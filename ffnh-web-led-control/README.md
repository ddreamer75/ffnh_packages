
# ffnh-web-led-control (Frontend)

Admin‑Seite unter „Advanced settings“ → „LED Control“.
- zeigt nur tatsächlich unterstützte **Farben**
- blendet **Helligkeit** aus, wenn keine dimmbare LED vorhanden ist
- **Trigger**: permanent/timer (bei timer: `delay_on/off`)
