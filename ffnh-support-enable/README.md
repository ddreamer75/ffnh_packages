# ffnh-support-enable

Dieses Gluon-Paket fügt im Konfigurationsmodus (Config Mode) eine Option hinzu, die es ermöglicht, den Supportzugang des **Freifunk Nordhessen e.V.** für einen Knoten zu aktivieren.

Wenn aktiviert, werden die in der `site.conf` hinterlegten Support-SSH-Keys automatisch hinzugefügt und bei Änderungen synchronisiert. Nutzerschlüssel bleiben unangetastet.
